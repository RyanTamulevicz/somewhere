Usage Example

<!-- Basic usage -->
<address-input country="US"></address-input>

<!-- With initial value -->
<address-input 
  country="CA"
  value='{
    "addressLine1": "100 Queen St", 
    "city": "Ottawa", 
    "administrativeArea": "ON", 
    "postalCode": "K1P 1J9"
  }'>
</address-input>

<!-- Show organization field -->
<address-input country="US" show-organization></address-input>

<script>
  const addressInput = document.querySelector('address-input');
  const isValid = addressInput.validate();

  console.log('Manual validation result:', isValid);
</script>

Features

  • ✅ 250+ countries supported - Based on Google's libaddressinput
  • ✅ Dynamic field visibility - Only shows fields required for each country
  • ✅ State/Province dropdowns - Where applicable (US, CA, AU, JP, etc.)
  • ✅ Postal code validation - Pattern matching per country
  • ✅ Manual validation trigger - Call validate() when you need it
  • ✅ No postal code countries - Hong Kong, Ireland, Jamaica, etc.
  • ✅ Proper terminology - State/Province/Prefecture/County based on country
  • ✅ Real-time validation - Events on change with validation status
